2025 Concluído
Origem — Especialização na UFG

O projeto nasce dentro da Especialização em Agentes e Sistemas Inteligentes da Universidade Federal de Goiás como trabalho de conclusão de curso. Desde o início, o objetivo vai além do TCC: construir algo que funcione de verdade e que possa ser usado por qualquer pessoa.

UFG Especialização TCC
✓
2025 Concluído
Especificação — Personas, arquitetura e casos de uso

Sete personas reais definidas para guiar cada decisão do sistema. Casos de uso documentados com critérios de aceitação. Arquitetura desenhada: agentes especializados, orquestração via LangGraph, integração via MCP, infraestrutura em contêineres. Todas as decisões arquiteturais relevantes registradas como ADRs públicas.

LangGraph MCP Docker Personas

Início de 2026 Concluído
Primeiros componentes em funcionamento

Orquestrador com LangGraph, interface web com Django, primeiras ferramentas MCP operacionais: consultar CNPJ, buscar processos, buscar notícias. O ciclo básico de inteligência — dado bruto → informação estruturada → análise — funcionando de ponta a ponta, localmente.

Abertura ao público — Site e manifesto

Site publicado, manifesto lançado, repositório público com documentação completa de arquitetura, personas, roadmap e decisões. O projeto sai do desenvolvimento privado e começa a existir publicamente. Primeiro passo para construir em aberto.

Agora Em andamento
Fase 0 — MVP local

Sistema funcionando localmente via docker-compose com PostgreSQL, MinIO e Qdrant. Agentes Planejador, Coletor e Redator operacionais. Motor de políticas bloqueando LLM externo para dados sensíveis. Caso de uso: investigação de empresa com dados públicos.

docker-compose PostgreSQL Qdrant MinIO
○
Próximo Planejado
Fase 1 — Dados privados e RAG

O usuário poderá trazer seus próprios documentos. Pipeline RAG completo: indexação, fragmentação, embedding, busca semântica. Agente Extrator com OCR. LLM local via Ollama para dados sigilosos — dados restritos nunca saem para APIs externas.

RAG Ollama OCR Privacidade
○
Próximo Planejado
Fase 2 — Agentes completos e grafo de vínculos

Correlacionador, Validador e Analista incorporados. Grafo de vínculos com Neo4j. Relatórios com fontes rastreáveis, grau de confiança por afirmação e distinção explícita entre fato, inferência e opinião. Exportação em PDF.

Neo4j Grafo Correlação
○
Próximo Planejado
Fase 3 — Interface de voz e chat natural

Qualquer cidadão consegue usar o sistema sem letramento digital. STT e TTS em português brasileiro. Conversa em múltiplos turnos. Resposta adaptada por persona — Dona Maria recebe uma resposta diferente da que o Auditor da Receita recebe para a mesma pergunta.

Visão de longo prazo Horizonte
Parceria com o Estado — inteligência como serviço público

O objetivo final: o Estado provê infraestrutura de LLM como serviço público — como provê educação e saúde. O Inteligência Aberta como camada de aplicação sobre essa infraestrutura, acessível a qualquer cidadão, gratuitamente. Esta fase depende de articulação política além do desenvolvimento técnico.
